# DevSearch
This is a <strong> Project sharing </strong> website made in <strong> Django </strong>.



# Features
<ul>
<li> Share Projects </li>
<li> Message other developers </li>
<li> Rate others work </li>
<li> Search other developers </li>
</ul>
